(Reference Link w3schools)

Cascading Style Sheets determine colors, locations, and extra effects on the tags on your web page. CSS is properly done with a .css file listing all css commands for your website.
I do CSS inside the tag. It overwrites all other methods.

Here is typical in-document CSS

  color: #00ff00;
}

T2A03: Classes

Find a good reference to Javascript Classes
Classes are both Objects that store variables (Called Properties) and Functions that work on the variables (called Methods)

Example Classses code

class myAnimal {
    constructor(passName, passAge) {
        this.myName = passName;
        this.myAge = passAge;
    }
    myShow(){    
        document.getElementById('myDiv01').innerHTML = 'Hello '+ this.myName + '. You are ' + this.myAge + ' years old'
    }
}
myDog = new myAnimal('Snickers', 5)
myDog.myShow()

A class I made that uses the constructor() and custom methods


class person {
    constructor(firstName, lastName, year, eyeColour) {
        this.firstName = firstName
        this.lastName = lastName
        this.date = (year - 1970) * 31536000000
        this.eyeColour = eyeColour
    }
    fullName() {
        return `${this.firstName} ${this.lastName}`
    }
    ageYear() {
        return (Date.now() - this.date) / 31536000000
    }
    ageSec() {
        return (Date.now() - this.date) / 1000
    }
}
